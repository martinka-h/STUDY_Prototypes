## Instructions:

1. Download Cinemachine.
2. Put CinemachineVirtualCamera prefab into the scene. The VirtualCamera script belongs on this object.
3. Put the CameraSystem prefab into the scene. The CameraSystem script belongs on this object.
4. Put the CameraSystem into the fields Follow and LookAt in CinemachineVirtualCamera.

The script on CinemachineVirtualCamera will add the components that are missing.
Follow the instructions in the console.
